# XHS Note-Style Guide

## Canvas
- Size: 1080 x 1920 (9:16)
- Safe margin: 96 px all sides
- Content width: 888 px

## Typography
- Title: 64-80 px, bold
- Subtitle/Highlight: 40-48 px, semi-bold
- Body bullets: 30-36 px, regular
- Recommended fonts: "PingFang SC", "Noto Sans SC", "Source Han Sans", "Hiragino Sans GB"

## Layout
- Large title top-left.
- One highlight block (soft background) near top.
- Bullet list with generous line height (1.35-1.5).
- Optional tag chips (2-3 max).
- Keep whitespace: avoid filling the whole page.

## Color System
- Background: off-white (#FFF8F2) or soft warm gray (#F7F7F5).
- Primary text: near-black (#1E1E1E).
- Accent: XHS-like red (#FF4D6D) or coral (#FF6B6B).
- Secondary: muted gray (#6B6B6B).

## Do / Don't
- Do use short punchy headlines.
- Do keep bullets short and scannable.
- Don't use heavy gradients or neon colors.
- Don't compress text to the edges.

## Style Variants
- Note-paper: lined background, brush highlight behind title, handwriting vibe.
- Bold card: dark gradient backdrop + light rounded panel, heavy bold text, red highlight words.

## Video Mode (Release Clips)
- For release videos, use premium gradient backgrounds instead of flat color.
- Keep all meaningful content in the centered middle 2/3 safe area.
- Section text should be card-like rows with clear spacing for mobile readability.
- Start each scene with visible content; avoid blank background-only opening frames.
- Default to one unified palette for the full video unless variation is requested.
